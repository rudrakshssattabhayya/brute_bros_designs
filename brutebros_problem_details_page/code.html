P1: Two Sums

Algorithm Challenge

Problem Statement

Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`. You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order. This section explains the task, defines the input and output formats, and lists any constraints that the solution must adhere to. The description will be clear and concise to ensure programmers fully understand the challenge before attempting to write code.

Input Examples

nums = [2,7,11,15], target = 9
nums = [3,2,4], target = 6
nums = [3,3], target = 6

Output Examples

[0,1]
[1,2]
[0,1]
